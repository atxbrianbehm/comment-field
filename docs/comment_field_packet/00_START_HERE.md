## Immediate objective

Build a local-first web application that can produce the Papa Murphy's social-comment sequence without requiring code edits:

- generate a deterministic comment field
- art-direct card positions and depth
- create random or mouse-directed build-ons
- promote one card as the hero
- locally reflow nearby cards
- save multiple takes
- export a deterministic preview or PNG sequence

Anything not required by the acceptance test should remain unimplemented in the first build.
